## Archive Organization

Each archived study maintains its complete research package:

```
archive/
├── [study_name]/
│   ├── README.md              # Study overview and findings
│   ├── protocol.md            # Research methodology
│   ├── data/                  # Observations, trials, artifacts
│   ├── analysis/              # Findings and synthesis
│   └── [implementation docs]  # If study led to system changes
```

**Nothing is deleted.** Complete research packages preserve:
- Decision-making rationale
- Alternative approaches considered
- Trial data and outcomes
- Evolution of understanding
- Dead ends explored

This transparency enables future researchers to:
- Understand why current practice is the way it is
- Learn from what didn't work
- Revisit conclusions with new information
- Build on solid foundation
